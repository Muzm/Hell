## 快速开始

1. `npm install @helldoc/cli -g` 安装 HellDoc
2. `echo '# Hello World' > README.md` 新建一个 README.md 文件
3. `helldoc dev` 启动

开始体验吧。

## 了解更多

运行 `helldoc -h` 看看吧

## 关于此项目

最初是看到了[`vuepress`](https://vuepress.vuejs.org/zh/)这个基于 vue 的优秀文档工具，想要从其中学习前端工具的一些开发技巧；
于是乎就基于`TypeScript`和`React`造了这个轮子。
